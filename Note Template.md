## Cross-Reference Methods

### 1. Direct Links
Use `[[Note Name]]` for specific related concepts:
- Example: `[[Docker Compose]]`

### 2. MOCs (Maps of Content)
Link to index notes for broad topics:
- Example: `[[Software Engineering/Cloud/AWS/Index]]`

### 3. Block References
Link to specific paragraphs using `[[Note^block-id]]`:
- Example: `[[API Design^rest-definition]]`

### 4. Embeds
Pull content into current note using `![[Note]]`:
- Example: `![[Code Snippet for Authentication]]`
