## 📝 Competency Matrix

### Summary
A competency matrix defines the skills, knowledge, and behaviours expected at each level of a role. It provides a shared language for career development, performance conversations, and hiring.

### Content

#### SFIA (Skills Framework for the Information Age)
- [SFIA 9](https://sfia-online.org/en/sfia-9) - Industry-standard framework defining IT and digital skills across seven levels of responsibility
